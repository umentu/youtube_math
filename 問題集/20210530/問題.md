### 次の関数を $x$ に関して微分しましょう。

1. $f(x) = 2x + 3$
1. $f(x) = 4$
1. $f(x) = 4x^2 + 10$
1. $f(x) = x ^ 2 + 3 x + 4$
1. $g(x) = (x + 3)(x + 4)$

### 次の関数のある点の接線の傾きを求めましょう。

1. $f(x) = 2 x + 3$ の $ x = 3$ における接線の傾き
1. $f(x) = 3 x^2 + 2x + 1$ の $ x = 1$ における接線の傾き
1. $f(x) = x^3 + 2x^2 + x + 4$ の $x = 0$ における接線の傾き


### 次の問題を解きましょう（やや応用）

$f(x) = x^2 + 2x + 1$ の $x = 2$ における接線の関数を式で表してください。
